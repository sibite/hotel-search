## Getting Started

### Prerequisites

- Node.js (18.14.0 for best compatibility)

### Preparation

1. Clone the repo

   ```sh
   git clone <repository_url>
   ```
   
2. Install NPM packages

   ```sh
   npm install
   ```

### Development

1. Start the React app

   ```sh
   npm start
   ```
